Polynomial Arithmetic

A polynomial adder/multiplier.

The Project

This project is a polynomial adder and multiplier. It reads two polynomial from standard input and either adds or multiplies them.

Lessons Learned

I learned how to handle very large numbers. At the conclusion of every project, we review our code using certain rules; in this case, we used the STR and MEM rules. I am more farmiliar with all CERT rules, but from this project, the aforementioned rules in particular.

GeeMail

A secure local email client.

The Project

GeeMail is a secure asynchronous (same-machine) message platform. It prompts the user to sign-in or register. Once logged in, the user can either read their received messages or write a new one. All data is stored in an SQLite database and important information (such as passwords) is enecrypted using AES. I used a simple version of Diffie-Hellman key exchange to establish a shared key between users and use the shared key to encrypt the information. That way, only real and appripriate users can decrypt and read the messages.

Lessons Learned

I have learned a great deal more about encrpytion and security from doing this project. It was important to modularize the code into different files and classes. This project has both a GeeMail class as well as a database class. Additionally, since this was the most demanding of the projects completed in the course, I had to make sure everyone in the group was on the same page and knew what had to be completed. Working with others is a crucial skill in the workplace and all projects, especially this one, helped with that regard.

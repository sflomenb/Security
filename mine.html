Minesweeper

A command-line version of the popular game.

The Project

This is a command-line version of Minesweeper in C++. When launching the game (using ./main 7 5 3 for example), it takes 3 parameters: the width, height and the number of mines.

Lessons Learned

This project further reinforced my C++ skills. It was more complex than the previus Markov Tweet. project, requiring me to split the project up into various files.

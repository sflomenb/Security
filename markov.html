Markovian Tweets

A pseduo-random tweet generator using Markov Sentence generation.

The Project

In genral, this project creates pseudo-random tweets using Markov Sentence generation.

Version 1: C++

This project was initially developed in C++. It reads samples of tweets from a .txt file and generates an approproate tweet. This project really helped relearn the fundamentals of C++.

Version 2: JavaScript

A second version of this project was developed using JavaScript. Unlike the previous version, this JavaScript version has a GUI and takes as input a user's Twitter handle.

Lessons Learned

This project was a great way to start the semester. I became more comfortable with C++, improving up the areas I knew, shoring up the areas that were weak and learning completely new things. Additionally, it was interesting to make a project in two different languages. I have learned when it is apporproate to use C++ or JavaScript. They are each suited for different things. If I want to build an interface and connect to the internet, JavaScript would be more appropriate, where if I wanted to really focus on security, C++ would be a better choice.
